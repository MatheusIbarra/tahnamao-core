# PR Governance Checklist

- [ ] Regras de negocio alteradas foram documentadas em `docs/business-rules`.
- [ ] Alteracoes de endpoint atualizaram `docs/contracts/openapi/openapi.yaml`.
- [ ] Collection Postman foi sincronizada em `docs/contracts/postman`.
- [ ] Decisoes arquiteturais relevantes geraram ADR.
- [ ] Modulo afetado teve `docs/modules/*.module.md` atualizado.
- [ ] Changelog tecnico do modulo foi atualizado.
- [ ] Testes unitarios e de contrato cobrem o comportamento novo.
